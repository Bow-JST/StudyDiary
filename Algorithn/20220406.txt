1.回顾第一章的知识点
（1）算法的概念：解决问题的一种方法或一个过程
（2）算法的四条性质：
输入：零个或多个外部的量作为算法的输入
输出：算法至少产生一个量作为外部的输出
确定性：组成算法的每条指令都是清晰的，无歧义的
有限性：算法的每条指令的执行次数是有限的，执行每条指令的时间也是有限的。
（3）算法的时间复杂度
平时常用的就是大O表示法

2.开启第二章分治算法

（1）引入分治的基本思想
将一个问题规模为n的问题分解为k个规模较小的子问题，这些子问题互相独立且与原问题相同。递归地解这些子问题，然后将个子问题的解合并得到原问题的解。

（2）利用JS演示了递归实现阶乘的算法。
```JavaScript
function factorial(){
    if(n == 1){
        return 1;
    }

    return n * factorial(n - 1);
}
```
